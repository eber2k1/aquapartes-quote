## ✨ Características Principales

### 💼 Para Vendedores
- **Catálogo Inteligente:** Búsqueda rápida, imágenes y fichas técnicas en PDF. Prevención de productos duplicados en tiempo real.
- **Cotizaciones Rápidas:** Creación de cotizaciones con cálculo automático de totales y generación de PDF oficial listo para compartir por WhatsApp o Correo.
- **Gestión de Clientes:** Importación automática desde la agenda de contactos del teléfono celular.
- **Modo Offline:** Posibilidad de acceder como invitado para ver los datos guardados en caché sin necesidad de internet.

### 👑 Para Administradores
- **Dashboard y Analítica:** Gráficas interactivas de ventas mensuales, distribución de estados y top de productos más vendidos.
- **Trazabilidad Total:** Registro automático del vendedor que crea cada cliente, producto o cotización.
- **Gestión de Empleados:** Creación, edición y desactivación de cuentas de vendedores desde la propia app.

## 🛠 Arquitectura y Tecnologías
- **Framework:** Flutter (Material 3).
- **Arquitectura:** Feature-First / Clean Architecture.
- **Gestión de Estado:** Provider / StateNotifier.
- **Red y Caché:** ApiClient personalizado con SharedPreferences.
- **Generación de PDFs:** Paquete `pdf` y `printing`.
- **Gráficos:** `fl_chart`.
